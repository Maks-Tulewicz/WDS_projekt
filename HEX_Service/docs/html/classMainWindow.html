Detailed Description

Główne okno aplikacji sterującej hexapodem.

Klasa odpowiada za wyświetlanie głównych widoków aplikacji oraz obsługę przycisków nawigacyjnych. Użytkownik może przełączać się pomiędzy widokiem połączenia FM, animacjami, ustawieniami i kątami serw.
